# Проект: Console Menu Manager (Python)

Простое консольное приложение на Python для управления меню кафе или ресторана. Проект демонстрирует базовые навыки бэкенд-разработки: работу с файловой системой, обработку исключений и реализацию логики CRUD (Create, Read, Delete).

## 🚀 Функционал
*   **Просмотр меню**: Чтение данных из текстового файла и вывод в консоль.
*   **Добавление позиций**: Интерактивный ввод новых блюд с сохранением в файл.
*   **Удаление позиций**: Выбор конкретной строки по номеру и автоматическое обновление файла.
*   **Валидация данных**: Защита от некорректного ввода (буквы вместо цифр) и обработка ошибок отсутствия файла.

## 🛠 Технологии
*   **Язык**: Python 3.x
*   **Хранение данных**: TXT-файл (используется как локальная база данных).

## 🧩 Ключевые концепции кода
В проекте реализованы важные принципы программирования:
1.  **Контекстные менеджеры (`with open`)**: Для безопасной работы с файлами без утечек памяти.
2.  **Режимы доступа**:
    *   `a` (Append) — для быстрого добавления данных.
    *   `w` (Write) — для полной перезаписи файла при удалении элементов.
3.  **Синхронизация индексов**: Реализован перевод между человеческим счетом (от 1) и компьютерным (от 0) с помощью `enumerate`.
4.  **Обработка исключений**: Использование блоков `try-except` для предотвращения аварийного завершения программы (`ValueError`, `FileNotFoundError`).
